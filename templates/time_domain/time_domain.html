# 时域指标 数学表达式 计算数值
1 最大值 `max(x)` {{ max_value }}
2 最小值 `min(x)` {{ min_value }}
3 平均值 `x = 1/N\sum_{i=1}^{n}x_n` {{ mean_value }}
4 均方根值 `x = \sqrt{1/N\sum_{i=1}^{n}x_n^2}` {{ root_mean_value }}
5 峰峰值 `pk = max(x)-min(x)` {{ pk_value }}
6 绝对平均值 `x = 1/N\sum_{i=1}^{n}\abs{x_n}` {{ av_value }}
7 峰值因子 `x = {pk}/{\sqrt{1/N\sum_{i=1}^{n}x_n^2}}` {{ crest_factor_value }}
8 波形因子 `x = \sqrt{1/N\sum_{i=1}^{n}x_n^2}/{1/N\sum_{i=1}^{n}\abs{x_n}}` {{ shap_factor_value }}
9 脉冲指标 `x = {pk}/{1/N\sum_{i=1}^{n}\abs{x_n}}` {{ impulse_factor_value }}
10 裕度指标 `x = {pk}/{1/N\sum_{i=1}^{n}\sqrt{\abs{x_n}}}^2` {{ clarance_value }}
11 峭度指标 `K = {\sum_{i=1}^n[x_n-\overline{x}]^4}/{(N-1)\sigma_x^4}` {{ kur_value }}
12 偏度 `S = {\sum_{i=1}^n[x_n-\overline{x}]^3}/{(N-1)\sigma_x^3}` {{ skew_value }}
13 方差 `\delta = 1/N\sum_{i=1}^{n}x_n^2` {{ var_value }}
